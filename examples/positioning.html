position: sticky;

keeps an element in a fixed position within its container when scrolling past a specified threshold, but allows it to move with the scroll until that threshold is reached.

Mostly used in : Sidebar, Header 

<!-- HTML -->
<section class="sticky-parent">
    <div class="wrapper">
        <div class="example-sidebar">
            <h5>Sidebar</h5>
            <ul>
                <li><a>Link 1</a></li>
                <li><a>Link 2</a></li>
                <li><a>Link 3</a></li>
                <li><a>Link 4</a></li>
                <li><a>Link 5</a></li>
                <li><a>Link 6</a></li>
            </ul>
        </div>
        <div class="content">
          ... Not sticky content ...
        </div>
    </div>
</section>

<!-- CSS -->
.sticky {
  .wrapper {
    display: flex;
    gap: 1.5rem;
  }

  .example-sidebar {
    position: sticky;
    top: 1rem; /* Adjust this value based on your header height or desired offset */
    flex: 0 0 300px; /* Flex-basis for the example-sidebar width */
  }

  .content {
    flex: 1;
  }
}
Example
position: fixed;

positions an element relative to the viewport, keeping it in place even when the page is scrolled.

Mostly used in : Modals 

<!-- HTML -->
<div class="popup">
    <div class="content">
        Hello World
    <div class="content">
</div>

<div class="toaster">
    Hello World
</div>

<!-- CSS -->
.popup {
    position: fixed;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
}

.toaster {
    position: fixed;
    bottom: 1.5rem;
    left: 1.5rem;
}
Example
position: absolute;

positions an element relative to its nearest positioned ancestor, or the initial containing block if none exists, removing it from the normal document flow.

Mostly used in : Inner component 

<!-- HTML -->
<button data-tooltip="Top opening" data-tooltip-pos="top">
    Example #1
</button>
<button data-tooltip="Bottom opening" data-tooltip-pos="bot">
    Example #2
</button>
<button data-tooltip="Right opening" data-tooltip-pos="right">
    Example #3
</button>
<button data-tooltip="Left opening" data-tooltip-pos="left">
    Example #4
</button>

<!-- CSS -->
[data-tooltip] {
  position: relative;
  cursor: pointer;

  &::after {
    content: attr(data-tooltip);
    position: absolute;
    left: 50%;
    background-color: #333;
    color: #fff;
    padding: 0.25rem 0.75rem;
    border-radius: 0.25rem;
    white-space: nowrap;
    opacity: 0;
    pointer-events: none;
    transition: opacity 0.2s ease-in-out;
    font-weight: 300;
    font-size: 0.85rem;
    transform: translate(-50%, -100%);
    top: -0.5rem;
  }

  &[data-tooltip-pos="bot"] {
    &::after {
      top: auto;
      bottom: -0.5rem;
      transform: translate(-50%, 100%);
    }
  }

  &[data-tooltip-pos="right"] {
    &::after {
      top: 50%;
      left: 100%;
      transform: translate(0.5rem, -50%);
    }
  }

  &[data-tooltip-pos="left"] {
    &::after {
      top: 50%;
      left: auto;
      right: 100%;
      transform: translate(-0.5rem, -50%);
    }
  }

  &[data-tooltip-pos="top"] {
    &::after {
      top: -0.5rem;
      bottom: auto;
      transform: translate(-50%, -100%);
    }
  }

  &:hover::after {
    opacity: 1;
  }
}
